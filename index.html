    ID226 Advanced Web Scripting

    Taught by Holly Silvia, this course builds on our existing knowledge of Javascript. Each week, we will test new scripting applications with example exercises, and then create custom projects to implement the new concepts.

    In addition to vanilla Javascript, we're also learning a popular Javascript library called jQuery. It helps to streamline things like things like HTML document traversal/manipulation, event handling, animation, etc. Finally, we will cover Node.js, a runtime environment and library for running web applications outside the client's browser.
