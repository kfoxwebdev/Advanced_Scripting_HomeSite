Web Storage App

This app lets users make or change a mad-libs story. Our javascript takes whatever the user types in each box box, and adds it to a local storage object. Additional submissions can be concatenated to each object, which are then displayed in the story at the bottom of the page. This allows users to submit multiple words in a row and see all of their submissions added to the story. The "clear" buttons run a scrpit that removes the local storage objects to reset the story.
